# Binary Electronegativity vs delta_E

This folder compares metal electronegativity with grouped binary `delta_E` statistics.

Separate plots are provided for:

- oxygen systems
- sulfur systems

Each ligand is shown for four statistics:

- `min`
- `max`
- `mean`
- `median`

## Why these plots matter

These figures test whether electronegativity alone provides a simple linear explanation for the binary `delta_E` trends. In practice, the strongest signal appears in the extreme statistics rather than in the mean or median values.

## Dataset size

- metals included: Co, Cr, Cu, Fe, Hf, Mn, Mo, Nb, Ni, Pd, Rh, Sc, Ta, Tc, Ti, V, Y, Zn, Zr
- grouped rows: `37`
