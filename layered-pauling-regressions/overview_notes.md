# Binary Electronegativity Multi-Regression Plots

This folder contains cleaner follow-up figures for discussing electronegativity trends in the binary dataset.

The purpose of these plots is to compare several regression lines at once without the clutter of point-by-point labels.

## Outputs

- `plots/binary_o_clean_multiple_regression.png`
- `plots/binary_s_clean_multiple_regression.png`
- `plots/binary_clean_statistic_comparison_grid.png`
- `tables/layered_regression_summary.csv`
- `drafting_outline.md`
- `pauling_discussion.md`

## What these figures show

The strongest electronegativity signal appears in the extreme statistics:

- oxygen `min delta_E`
- sulfur `max delta_E`

By contrast, the `mean` and `median` regressions are weak for both ligands, which suggests that electronegativity is not the main descriptor for the overall binary response.
