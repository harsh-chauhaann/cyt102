# Bar-Graph Statistics

This folder collects grouped `delta_E` summaries shown as bar charts.

Grouping scheme:

- binary: `(metal, ligand)`
- ternary: `(metal pair, ligand)`

Statistics included:

- `min`
- `max`
- `mean`
- `median`

Each plot prints the numerical value near the bar tip so the ranking is easy to compare visually.

## Why these plots matter

The bar charts are the quickest way to identify which systems dominate the negative tail and how that ordering changes between worst-case, average, and best-case views.

## Dataset size

- binary grouped rows: `37`
- ternary grouped rows: `198`
